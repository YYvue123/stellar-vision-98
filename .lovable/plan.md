

## 探索卡片骨架屏加载状态

### 方案

为 `ExploreCards` 组件添加 `isLoading` prop，当加载中时显示骨架屏替代真实内容，为未来接入 API 做准备。

### 改动

**`src/components/ai-audio/ExploreCards.tsx`**
- Props 新增 `isLoading?: boolean`
- 新增 `ExploreCardsSkeleton` 内部组件，包含：
  - 欢迎区域：两行 Skeleton 文字占位
  - 探索区域：5 个卡片骨架（正方形封面 + 标题行 + 类型行），响应式布局与真实卡片一致（移动端横排，桌面端竖排）
  - 功能卡片区域：4 个矩形骨架
- 当 `isLoading` 为 true 时渲染骨架屏，否则渲染原有内容
- 使用已有的 `Skeleton` 组件（`src/components/ui/skeleton.tsx`）

**`src/pages/AIAudio.tsx`**
- 传入 `isLoading={false}` 给 ExploreCards（当前硬编码数据不需要加载，但预留接口）
